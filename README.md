# Homecare Technicians | Professional Appliance Repair

Expert home appliance repair services in Faridabad and South Delhi. Fast, reliable, and guaranteed repairs for all major brands.

## Features

- **Professional Services:** AC, Washing Machine, Refrigerator, LED TV, Microwave, and more.
- **Trusted Experts:** 15+ years of experience.
- **Transparency:** Fixed pricing and no hidden charges.
- **Convenience:** Same-day service at your doorstep.
- **Warranty:** 90-day service warranty on repairs.

## Tech Stack

- **Frontend:** HTML5, Tailwind CSS 4, Alpine.js
- **Build Tool:** Vite
- **Deployment:** GitHub Pages (via GitHub Actions)

## Development

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run locally:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Deployment

To deploy the website to GitHub Pages, run:

```bash
npm run publish:github
```

This script will:
1. Ensure the repository is initialized and connected to GitHub.
2. Configure the GitHub Actions workflow for deployment.
3. Push the source code to the `main` branch.
4. Trigger the build and deployment process.
